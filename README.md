# Expense Tracker Application

A simple Expense Tracker Application built using JavaScript.

## Features

* Add Expenses
* View Expense List
* Delete Expenses
* Calculate Total Expenses
* Console-Based Application

## Technologies Used

* JavaScript (ES6)
* Node.js

## How to Run

1. Clone the repository
2. Open the project folder in VS Code
3. Open Terminal
4. Run:

```bash
node expenseTracker.js
```

## Concepts Used

* Arrays
* Objects
* Functions
* forEach()
* reduce()
* splice()
