# Counter App

A simple Counter Application built using HTML, CSS, and JavaScript.

## Project Overview

This project allows users to:

* Increase the counter value
* Decrease the counter value
* Reset the counter to zero

The application demonstrates basic JavaScript concepts such as variables, functions, DOM manipulation, and event handling.

## Features

* Increment Counter
* Decrement Counter
* Reset Counter
* Responsive and User-Friendly Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
counter-app/
│
├── index.html
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Use the buttons to increase, decrease, or reset the counter.
